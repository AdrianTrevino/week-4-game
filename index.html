Crystals Collector!








You have a random number to guess at the start of the game!

Out of the four crystals below, each one has a certain point when you click.

You win if you can match your number to the randomly guessed number from the beginning. You lose if you go over that randomly guessed number.

The value of each crystal is hidden. You can click on it to find out the amount of points it's worth.

Each time you start the game, the value of the crystals change. No body said it was going to be easy! Good luck!!
